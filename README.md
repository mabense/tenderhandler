# Tenderhandler

## Assignment

### Goal
A learning project about tracking the fulfillment and payout of tenders.

### Required Features
* For All Users
  * sign up
  * log in/out
  * read data (R)
* For Administrators
  * insert data (C)
  * read data (R)
  * modify data (U)
  * delete data (D)
* For Managers
  * modify data (U)
